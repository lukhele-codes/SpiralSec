# SpiralSec
SpiralSec is a modular C# console chatbot designed to educate users about cybersecurity awareness.
It combines voice greeting, ASCII art branding, and an enhanced console UI to create an engaging, interactive experience.
The chatbot responds to user queries about safe online practices, passwords, phishing, and general cybersecurity topics.

 Objectives
Deliver a professional, modular C# application.
Implement voice and visual elements for immersion.
Use colored text, borders, and typing effects to enhance the console UI.
Demonstrate GitHub version control and CI/CD setup through meaningful commits.

Class Structure
Class	Responsibility
Play	Plays the voice greeting (SpiralSecAudio.wav).
Spiral	Displays ASCII logo branding.
User	Welcomes the user and captures their name.
AIbot	Handles responses and conversation logic.
Program	Orchestrates the overall chatbot flow.
Some of the features include
Voice Greeting: Plays a WAV file when the chatbot starts.
ASCII Logo: Displays a branded SpiralSec logo.
Personalized Interaction: Greets the user by name and maintains it throughout the session.
Colored text for roles and messages.
Knowledge Base: Responds to cybersecurity topics such as passwords, phishing, VPNs, and safe browsing.

How to Run
Open the project in Visual Studio.

Ensure SpiralSecAudio.wav and logo image are set to:

Build Action: Content

Copy to Output Directory: Copy always

Build and run the project (Ctrl + F5).

Interact with the chatbot in the console window.

Type exit to end the session.

For example:
Bohlale: What can I do?
AIbot: Bohlale, you can ask me about safe passwords, phishing, browsing securely, using VPNs, and general cybersecurity tips.

 Conclusion
SpiralSec demonstrates a professional, modular approach to console application development.
